BUILD INSTRUCTIONS

To make the installer script executable:

chmod +x ./install.sh

Run installer script:

./install.sh

If you do not have CMake installed, get it with:

sudo apt-get install cmake

SAMPLES

C++ samples are built by the installer script. Source code available in samples/C++

MATLAB samples are available in samples/MATLAB
